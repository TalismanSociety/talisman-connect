# @talisman-connect/nft

## Setup:

```
npm i --save @talisman-connect/nft
```

## Hooks

### `useNftsByAddress`

Retrieves all NFTs fron the given address

### `useNfts`

Retrieves all NFTs from all of the given addresses

### `useContentType`

Retrieves the URL's `content-type` header value.
